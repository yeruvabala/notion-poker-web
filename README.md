# Notion Poker Web (flat)

Deploy-ready. No extra folder level.

## Quick Deploy (Vercel)
1. Create **New Project** → **Add files** → upload this ZIP.
2. Framework preset: Next.js (auto).
3. Env vars:
   - OPENAI_API_KEY
   - NOTION_TOKEN
   - NOTION_DATABASE_ID
4. Deploy and open the URL.

## Local Dev
npm install
npm run dev
